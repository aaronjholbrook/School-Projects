# CPTS 322 Software Engineering Principles 1 & CPTS 451 Introduction to Database Systems.
A project that was shared between two classes where class and Entity Relationship diagrams plus the frontend were developed for one class, and the database backend in SQL (In 
PostgresSQL) for the other. The data was a set of Yelp data from an old coding challenge and the frontend used it to "login" and then view locations to see their friends
tips/reviews and place their own.
